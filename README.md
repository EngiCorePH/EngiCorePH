## 📚 EngiCore Review Hub

A complete ME Board Exam study companion — fully offline, no account required.

**Features:**
- 📊 Study dashboard with daily progress tracking
- 📐 50+ formulas across 8 ME subjects — searchable and filterable
- 📖 40+ key term definitions — board exam ready
- 📅 Daily checklist + weekly planner with progress bars
- ⏱️ Pomodoro timer with session log and 4-session cycle
- 🌙 Light / Dark / System theme toggle
- 💾 Data saved locally — works fully offline

**Subjects covered:**
Thermodynamics · Fluid Mechanics · Mechanics of Materials · Dynamics · Machine Design · Engineering Mathematics · Engineering Economics · Materials Science

## ⚙️ EngiCore Toolkit

Six engineering calculators in a single browser file — no install, no account.

**Calculators included:**
- ⚙️ Unit Converter — 8 categories (length, mass, force, pressure, power, energy, torque, temperature)
- 📐 Beam Deflection — 4 beam types with live diagram, deflection, moment, shear, and reactions
- ⚙️ Gear & Power — gear ratio, output speed, torque, kW, and horsepower with efficiency
- 🔩 Stress & Strain — normal stress, strain, elongation, safety factor, torsional shear
- 💧 Fluid Mechanics — Reynolds number, flow regime, Darcy-Weisbach head loss, pressure drop
- 📋 Formula Sheet — Statics, Thermo, Fluids, Dynamics, MoM, Machine Design

## 🛠️ Tech Stack

**Current (Free Tools)**
- Pure HTML, CSS, JavaScript — zero frameworks, zero dependencies
- Google Fonts (Syne + JetBrains Mono)
- `localStorage` for task, progress, and preference persistence
- GitHub Pages for free hosting
- Fully responsive — desktop and mobile

**Planned (Pro Version)**
- Supabase — authentication and PostgreSQL database
- Vercel — serverless deployment for authenticated app
- Stripe / GCash / Maya — payments for Philippine users
